# Security

Do not commit real credentials, tokens, email app passwords, NotebookLM browser profiles, Cloudflare secrets, Discord bot tokens, generated audio, logs, local databases, or personal recipient lists.

Expected private files include:

- `notebooklm-briefing-pipeline/config.json`
- `.env`, `.dev.vars`, and other local secret files
- `*.db`, logs, caches, and generated audio/briefing outputs

Use the included example config files as templates only.
